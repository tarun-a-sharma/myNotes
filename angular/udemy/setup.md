# Install the angular

-    First download [nodejs](https://nodejs.org/en/download)
- Install nodejs
- Then open system terminal and run the below command

    ```
    $ npm install -g @angular/cli
    ```

# Create new project

- Go to the folder where we want to create angular project
- Open terminal with that folder path.
- Run command

    ```
    ng new <angular-project-name>

    ng new first-angular-app
    ```
- Hit enter and select options.
- Go into the project folder in command prompt, hit below command to install all the dependencies.
    ```
    npm install
    ```

# Run Project

- Open integrated terminal with the folder path.
- Hit command 
    ```
    npm start
    ```
